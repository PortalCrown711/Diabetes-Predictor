# 🩺 Diabetes Prediction App

🔗 **Live App:** https://diabetes-predictor-prince.streamlit.app/

---

## 📌 Overview

This project is a **Machine Learning-powered web application** that predicts the likelihood of diabetes in a patient based on medical parameters.

The app is built using **Logistic Regression** and deployed online using **Streamlit Cloud**, providing an interactive dashboard for real-time predictions.

---

## 🎯 Features

* 📊 Predict diabetes risk (High / Low)
* 📈 Probability score (%)
* 📉 Visualization chart (Diabetes vs No Diabetes)
* 📋 Structured patient report
* 📥 Downloadable report (CSV format)
* 🎯 Demo profiles (Age 25–80) for quick testing
* 💻 Clean and responsive UI

---

## 🧠 Machine Learning Model

* Algorithm: Logistic Regression
* Dataset: Pima Indians Diabetes Dataset
* Type: Binary Classification
* Evaluation Metrics:

  * Accuracy
  * Confusion Matrix
  * ROC Curve

---

## 📊 Input Features

The model uses the following inputs:

* Pregnancies
* Glucose Level
* Blood Pressure
* Skin Thickness
* Insulin
* BMI (Body Mass Index)
* Diabetes Pedigree Function
* Age

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/PortalCrown711/diabetes-predictor.git
cd diabetes-predictor
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the app

```bash
streamlit run app.py
```
